# BudgetBuddy - Project Overview

## Purpose
BudgetBuddy is an F# full-stack web application that synchronizes bank transactions from Comdirect (German bank) to YNAB (You Need A Budget). It automates the process of importing transactions, detecting duplicates, and applying categorization rules.

## Key Features
- Comdirect API integration with Push-TAN authentication
- YNAB API integration for budget management
- Automatic duplicate detection
- Rule-based transaction categorization
- Transaction review workflow before import
- Real-time sync session management

## Target Users
Personal finance users who use both Comdirect banking and YNAB budgeting software.
